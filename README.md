# ChatAPT Chatbot

**Live Demo:** https://chatapt-chatbot.onrender.com/

Minimal AI chatbot interface using the Groq API.

## Setup

```powershell
git clone https://github.com/Jagadheeeeesh/ChatAPT-Chatbot.git
cd ChatAPT-Chatbot
```

## Usage

Type a message, press **Send**, and view AI responses.
